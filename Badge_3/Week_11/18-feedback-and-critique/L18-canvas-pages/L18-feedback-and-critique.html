Lesson 18: Feedback and Critique

🚀 Deliver Badge 💬 Module 3B

This lesson focuses on delivery, not just diagnosis. You already know how to find accessibility issues. Now the question is how to communicate them so other designers understand the problem, trust the feedback, and act on it.

Strong accessibility critique is specific, actionable, user-focused, and calibrated to the audience. Weak critique creates confusion or defensiveness, even when the finding itself is correct.

💡 Key Concepts

Constructive feedback 3B.01

Good accessibility feedback is specific, actionable, user-focused, and kind. It explains what the issue is, why it matters for users, and what should happen next.

Two useful frameworks are SBI (Situation-Behavior-Impact) and COIN (Context-Observation-Impact-Next). Use whichever structure improves clarity.

✓ Checkpoint: Constructive feedback

  • I can distinguish useful critique from vague criticism
  • I can connect findings to user impact

Critique facilitation 3B.02

A strong accessibility critique starts with scope, moves through findings in priority order, stays anchored in user impact, and ends with clear next steps.

Psychological safety matters because it keeps the session about the work rather than the person. De-escalation usually means re-anchoring on the affected users and the specific barrier under discussion.

✓ Checkpoint: Critique facilitation

  • I can describe a short critique structure
  • I know how to keep priority issues visible

Audience adaptation 3B.03

The same issue should be explained differently to a Discover Badge designer, a Define Badge designer, and a designer with no DACE background. The recommendation should remain clear even when the explanation changes.

Mentoring moments build understanding. Review moments evaluate the current work and define action. Badge 3 leaders need to know which mode they are in.

✓ Checkpoint: Audience adaptation

  • I can adapt feedback to different skill levels
  • I understand the difference between mentoring and review moments
