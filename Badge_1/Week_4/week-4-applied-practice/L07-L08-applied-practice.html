Week 4 Applied Practice: Navigation Accessibility Audit

Landmarks, Navigation Consistency & Skip Links Project

🔍 Discover Badge 🏷️ Module 1G 📝 16 Points

This applied practice project brings together the navigation skills you learned in Week 4: landmark regions, consistent navigation, and skip links. These skills ensure users can navigate efficiently with screen readers and keyboards, jumping directly to major page sections and bypassing repeated content.

You will work through an interactive workbook that guides you through analyzing the Northstar Shop design mockup. The workbook includes step-by-step exercises with immediate feedback and auto-grading.

🎯 What You'll Demonstrate

Landmarks & Structure

  • Identify all landmark regions (banner, navigation, main, region, contentinfo, search)
  • Assign unique accessible names when multiple landmarks of the same type exist
  • Match correct HTML5 semantic equivalents for each landmark
  • Document landmark structure across all pages

Skip Links & Navigation

  • Specify skip link destination and first tab target after skip link activation
  • Verify navigation consistency across all pages
  • Flag any inconsistencies in navigation order
  • Provide specific recommendations for fixing issues

Motion Accessibility

  • Inventory all motion and animation elements across the site
  • Assess WCAG compliance for each motion element
  • Identify which elements need pause/stop controls
  • Recommend reduced motion alternatives

💡 Tips for Success

Landmarks

  • Six landmark types: banner, navigation, main, region (complementary), contentinfo, search
  • One main per page: Every page should have exactly one main landmark
  • Name duplicates: Two nav elements need labels like "Primary navigation" and "Footer navigation"
  • Match HTML equivalents: The workbook auto-grades HTML5 semantic elements (e.g., banner = <header>)

Skip Links & Navigation

  • First focusable element: Skip link must be the very first thing a keyboard user can Tab to
  • Track focus carefully: After skip link activates and you Tab once, document exactly where focus lands
  • Navigation order: Check that navigation items appear in the same sequence on all pages
  • Use the mockup: The interactive mockup shows numbered zones and focus targets to guide your work

Motion Accessibility

  • Find all motion: Look for carousels, animations, parallax, auto-play videos, and transitions
  • Three-second rule: Motion lasting more than 5 seconds must have pause/stop controls
  • Essential vs decorative: Essential motion (like loading spinners) may have different requirements
  • Reduced motion: Consider how each motion element should behave when users prefer reduced motion

📜 Related WCAG Success Criteria

  • 2.4.1 Bypass Blocks (Level A): Users can skip repeated content via skip links, landmarks, or headings
  • 2.2.2 Pause, Stop, Hide (Level A): Motion lasting more than 5 seconds must have pause/stop controls
  • 2.3.3 Animation from Interactions (Level AAA): Motion triggered by interaction can be disabled (except essential motion)
  • 3.2.3 Consistent Navigation (Level AA): Navigation appears in same order across pages
