<h2 style="margin: 0 0 12px 0; font-size: 28px; color: #212529;">Lesson 08: Motion &amp; Animation</h2>
<div style="display: flex; flex-wrap: wrap; gap: 14px; margin-top: 18px;"><span style="display: inline-flex; align-items: center; gap: 8px; background-color: #e6fcf5; color: #087f5b; padding: 10px 18px; border-radius: 24px; font-size: 15px;"> <span style="font-size: 18px;">🔍</span> Discover Badge </span> <span style="display: inline-flex; align-items: center; gap: 8px; background-color: #e7f5ff; color: #1864ab; padding: 10px 18px; border-radius: 24px; font-size: 15px;"> <span style="font-size: 18px;">🎬</span> Module 1H </span></div>
</div>
<p style="font-size: 17px; color: #495057; margin-bottom: 16px; padding: 0 4px;">Motion and animation can enhance user experience, but they can also cause real harm. Users with vestibular disorders experience dizziness and nausea from parallax effects and animated transitions. Flashing content can trigger seizures. Auto-playing videos and carousels distract users with attention-related disabilities.</p>
<p style="font-size: 17px; color: #495057; margin-bottom: 32px; padding: 0 4px;">In this lesson, you'll learn to design motion that respects user preferences, implement pause controls for auto-playing content, avoid seizure-inducing patterns, and create reduced-motion alternatives that preserve meaning.</p>

<p style="margin: 0 0 20px 0; color: #495057; font-size: 15px;">Learn how motion on screen affects users with vestibular disorders and seizure conditions.</p>

<h4 style="margin: 0 0 12px 0; font-size: 17px; color: #212529; padding-bottom: 8px; border-bottom: 2px solid #d0ebff;">Knowledge Objectives</h4>
<ul style="margin: 0 0 28px 0; padding-left: 24px; color: #212529; font-size: 14px;">
<li style="margin-bottom: 10px;">Describe how motion on screen can trigger vestibular disorders (dizziness, nausea), seizures, or distraction for users with cognitive disabilities.</li>
<li style="margin-bottom: 10px;">Explain the 5-second rule: any automatic movement must stop within 5 seconds or provide a pause mechanism.</li>
<li style="margin-bottom: 10px;">Describe the difference between essential motion (communicates meaning) and decorative motion (purely aesthetic).</li>
<li style="margin-bottom: 10px;">Explain the prefers-reduced-motion media query and how it honors user device settings to limit or disable animations.</li>
<li style="margin-bottom: 10px;">Describe what constitutes a seizure risk: flashing content more than 3 times per second in a large enough area.</li>
<li>Explain WCAG 2.2.2 (Pause, Stop, Hide) and 2.3.1 (Three Flashes or Below Threshold) requirements.</li>
</ul>
<h4 style="margin: 0 0 12px 0; font-size: 17px; color: #212529; padding-bottom: 8px; border-bottom: 2px solid #d0ebff;">Skills Objectives</h4>
<ul style="margin: 0 0 28px 0; padding-left: 24px; color: #212529; font-size: 14px;">
<li style="margin-bottom: 10px;">Test designs against the Adobe Design Accessibility Checklist motion section with 90%+ accuracy.</li>
<li style="margin-bottom: 10px;">Identify animations, auto-playing videos, carousels, or parallax effects that exceed 5 seconds without a pause control.</li>
<li style="margin-bottom: 10px;">Spot content that flashes or blinks rapidly and flag potential seizure risks.</li>
<li>Verify designs specify how animations should behave when prefers-reduced-motion is enabled.</li>
</ul>
<h4 style="margin: 0 0 12px 0; font-size: 17px; color: #212529; padding-bottom: 8px; border-bottom: 2px solid #d0ebff;">Application Objectives</h4>
<ul style="margin: 0; padding-left: 24px; color: #212529; font-size: 14px;">
<li style="margin-bottom: 10px;">Flag motion issues during design reviews using the Adobe Design Accessibility Checklist.</li>
<li style="margin-bottom: 10px;">Annotate designs to indicate pause controls, reduced-motion alternatives, and animation durations.</li>
<li>Communicate the impact of problematic motion to team members in clear, non-technical language.</li>
</ul>

<div style="background-color: #f8f9fa; border: 1px solid #dee2e6; border-radius: 8px; padding: 24px; margin-bottom: 32px; border-left: 5px solid #0ca678;">
<h3 style="margin: 0 0 16px 0; font-size: 20px; color: #212529; display: flex; align-items: center; gap: 10px;"><span style="font-size: 22px;">📜</span> Related WCAG Success Criteria</h3>
<ul style="margin: 0; padding-left: 24px; color: #212529;">
<li style="margin-bottom: 8px;"><strong>2.2.2 Pause, Stop, Hide (Level A):</strong> Moving, blinking, or auto-updating content must be pausable, stoppable, or hideable</li>
<li style="margin-bottom: 8px;"><strong>2.3.1 Three Flashes or Below Threshold (Level A):</strong> Content must not flash more than 3 times per second</li>
<li><strong>2.3.3 Animation from Interactions (Level AA):</strong> Motion triggered by user interaction can be disabled unless essential</li>
</ul>
</div>
<div style="background-color: #fff; border: 1px solid #dee2e6; border-radius: 8px; padding: 24px; margin-bottom: 32px; border-left: 5px solid #f59f00;">
<h3 style="margin: 0 0 20px 0; font-size: 20px; color: #212529; display: flex; align-items: center; gap: 10px;"><span style="font-size: 22px;">💡</span> Key Concepts</h3>
<div style="margin-bottom: 28px;">
<h4 style="margin: 0 0 12px 0; font-size: 17px; color: #212529; padding-bottom: 8px; border-bottom: 2px solid #fff3bf; display: flex; align-items: center; gap: 10px;">Why Motion Accessibility Matters <span style="background-color: #d0ebff; color: #1864ab; padding: 2px 8px; border-radius: 4px; font-size: 11px; font-family: monospace;">ALL</span></h4>
<p style="margin: 0 0 16px 0; color: #212529;">Motion on screen can cause real physical harm. Unlike many accessibility issues that create inconvenience, motion problems can trigger medical conditions.</p>
<p style="margin: 0 0 8px 0; color: #212529;"><strong>Who is affected:</strong></p>
<ul style="margin: 0 0 16px 0; padding-left: 24px; color: #212529;">
<li style="margin-bottom: 6px;"><strong>Vestibular disorders:</strong> Parallax scrolling and animated transitions cause dizziness, nausea, and disorientation</li>
<li style="margin-bottom: 6px;"><strong>Seizure disorders:</strong> Flashing content can trigger photosensitive epileptic seizures</li>
<li style="margin-bottom: 6px;"><strong>Attention disabilities:</strong> Auto-playing videos and carousels distract from primary content</li>
<li><strong>Cognitive disabilities:</strong> Complex animations increase cognitive load and confusion</li>
</ul>
<div style="background-color: #ffe3e3; border-radius: 6px; padding: 14px 16px; border-left: 4px solid #fa5252;">
<p style="margin: 0; font-size: 14px; color: #212529;"><strong>Critical:</strong> Motion-triggered seizures can be life-threatening. This is not about user preference. It is about safety.</p>
</div>
</div>
<div style="margin-bottom: 28px;">
<h4 style="margin: 0 0 12px 0; font-size: 17px; color: #212529; padding-bottom: 8px; border-bottom: 2px solid #fff3bf; display: flex; align-items: center; gap: 10px;">The 5-Second Rule <span style="background-color: #d0ebff; color: #1864ab; padding: 2px 8px; border-radius: 4px; font-size: 11px; font-family: monospace;">1H.02</span> <span style="background-color: #e7f5ff; color: #1864ab; padding: 2px 8px; border-radius: 4px; font-size: 11px;">SC 2.2.2</span></h4>
<div style="background-color: #fff3bf; border-radius: 6px; padding: 14px 16px; border-left: 4px solid #f59f00; margin-bottom: 16px;">
<p style="margin: 0; font-size: 14px; color: #212529;"><strong>Key rule:</strong> Any automatic movement that lasts more than 5 seconds must have a mechanism to pause, stop, or hide it.</p>
</div>
<p style="margin: 0 0 16px 0; color: #212529;">SC 2.2.2 (Pause, Stop, Hide) applies to content that:</p>
<ul style="margin: 0 0 16px 0; padding-left: 24px; color: #212529;">
<li style="margin-bottom: 6px;"><strong>Starts automatically</strong> (not triggered by user action)</li>
<li style="margin-bottom: 6px;"><strong>Lasts more than 5 seconds</strong></li>
<li><strong>Appears alongside other content</strong> (not the only thing on screen)</li>
</ul>
<p style="margin: 0 0 8px 0; color: #212529;"><strong>Common examples requiring pause controls:</strong></p>
<ul style="margin: 0 0 16px 0; padding-left: 24px; color: #212529;">
<li style="margin-bottom: 6px;">Auto-playing video backgrounds</li>
<li style="margin-bottom: 6px;">Carousels and slideshows</li>
<li style="margin-bottom: 6px;">Animated illustrations or mascots</li>
<li style="margin-bottom: 6px;">Parallax scrolling effects</li>
<li>Loading animations (if longer than 5 seconds)</li>
</ul>
<div style="display: flex; flex-wrap: wrap; gap: 16px; margin-bottom: 16px;">
<div style="flex: 1; min-width: 250px; background-color: #d3f9d8; border-radius: 6px; padding: 16px;">
<h5 style="margin: 0 0 10px 0; font-size: 14px; color: #1e7b34;">✓ Compliant</h5>
<ul style="margin: 0; padding-left: 18px; color: #212529; font-size: 13px;">
<li>Visible pause/play button</li>
<li>Animation stops within 5 seconds</li>
<li>User-initiated animation only</li>
</ul>
</div>
<div style="flex: 1; min-width: 250px; background-color: #ffe3e3; border-radius: 6px; padding: 16px;">
<h5 style="margin: 0 0 10px 0; font-size: 14px; color: #a61e1e;">✗ Failure</h5>
<ul style="margin: 0; padding-left: 18px; color: #212529; font-size: 13px;">
<li>Infinite looping with no controls</li>
<li>Pause only works for current page</li>
<li>Controls hidden or hard to find</li>
</ul>
</div>
</div>
</div>
<div style="margin-bottom: 28px;">
<h4 style="margin: 0 0 12px 0; font-size: 17px; color: #212529; padding-bottom: 8px; border-bottom: 2px solid #fff3bf; display: flex; align-items: center; gap: 10px;">Reduced Motion Preferences <span style="background-color: #d0ebff; color: #1864ab; padding: 2px 8px; border-radius: 4px; font-size: 11px; font-family: monospace;">1H.01</span> <span style="background-color: #e7f5ff; color: #1864ab; padding: 2px 8px; border-radius: 4px; font-size: 11px;">SC 2.3.3</span></h4>
<p style="margin: 0 0 16px 0; color: #212529;">Users can set a "reduce motion" preference in their operating system settings. Designs must honor this preference using the <code style="background-color: #f1f3f4; padding: 2px 6px; border-radius: 3px; font-size: 13px;">prefers-reduced-motion</code> media query.</p>
<div style="background-color: #f8f9fa; border-radius: 6px; padding: 16px; margin-bottom: 16px; font-family: 'SF Mono', Monaco, monospace; font-size: 13px; overflow-x: auto;">
<code style="color: #212529;">@media (prefers-reduced-motion: reduce) {<br />&nbsp;&nbsp;/* Disable or reduce animations */<br />&nbsp;&nbsp;.animated-element { animation: none; }<br />}</code>
</div>
<p style="margin: 0 0 8px 0; color: #212529;"><strong>When reduced motion is enabled:</strong></p>
<ul style="margin: 0 0 16px 0; padding-left: 24px; color: #212529;">
<li style="margin-bottom: 6px;">Replace animations with instant state changes</li>
<li style="margin-bottom: 6px;">Disable parallax scrolling effects</li>
<li style="margin-bottom: 6px;">Stop auto-playing videos (or show static poster)</li>
<li>Use simple fades instead of complex transitions</li>
</ul>
<div style="background-color: #d0ebff; border-radius: 6px; padding: 14px 16px; border-left: 4px solid #1971c2;">
<p style="margin: 0; font-size: 14px; color: #1864ab;"><strong>Designer's role:</strong> Document in your specs how each animation should behave when prefers-reduced-motion is enabled. Do not leave this for engineering to guess.</p>
</div>
</div>
<div style="margin-bottom: 28px;">
<h4 style="margin: 0 0 12px 0; font-size: 17px; color: #212529; padding-bottom: 8px; border-bottom: 2px solid #fff3bf; display: flex; align-items: center; gap: 10px;">Seizure Risk and Flash Thresholds <span style="background-color: #d0ebff; color: #1864ab; padding: 2px 8px; border-radius: 4px; font-size: 11px; font-family: monospace;">1H.02</span> <span style="background-color: #e7f5ff; color: #1864ab; padding: 2px 8px; border-radius: 4px; font-size: 11px;">SC 2.3.1</span></h4>
<div style="background-color: #ffe3e3; border-radius: 6px; padding: 14px 16px; border-left: 4px solid #fa5252; margin-bottom: 16px;">
<p style="margin: 0; font-size: 14px; color: #212529;"><strong>Hard limit:</strong> Content must not flash more than 3 times per second. This threshold exists because flashing can trigger seizures in people with photosensitive epilepsy.</p>
</div>
<p style="margin: 0 0 8px 0; color: #212529;"><strong>What counts as a "flash":</strong></p>
<ul style="margin: 0 0 16px 0; padding-left: 24px; color: #212529;">
<li style="margin-bottom: 6px;">Rapid alternation between light and dark</li>
<li style="margin-bottom: 6px;">Strobe effects or blinking elements</li>
<li style="margin-bottom: 6px;">Video content with rapid scene changes</li>
<li>Animated GIFs with quick frame transitions</li>
</ul>
<p style="margin: 0 0 8px 0; color: #212529;"><strong>Area matters:</strong></p>
<p style="margin: 0 0 16px 0; color: #212529;">The flash threshold applies when the flashing area is large enough: more than 25% of 10 degrees of visual field (roughly 341x256 pixels at typical viewing distance). Small, isolated flashing elements may be acceptable, but always err on the side of caution.</p>
<div style="background-color: #fff3bf; border-left: 4px solid #fab005; padding: 16px;">
<p style="margin: 0 0 8px 0; color: #212529; font-size: 14px;"><strong>Red flash is especially dangerous</strong></p>
<p style="margin: 0; color: #212529; font-size: 14px;">Saturated red flashing is particularly likely to trigger seizures. Avoid red strobing effects entirely.</p>
</div>
</div>
<div>
<h4 style="margin: 0 0 12px 0; font-size: 17px; color: #212529; padding-bottom: 8px; border-bottom: 2px solid #fff3bf; display: flex; align-items: center; gap: 10px;">Designing Motion Alternatives <span style="background-color: #d0ebff; color: #1864ab; padding: 2px 8px; border-radius: 4px; font-size: 11px; font-family: monospace;">1H.03</span></h4>
<p style="margin: 0 0 16px 0; color: #212529;">When animation communicates meaning, you need a reduced-motion alternative that preserves that meaning without the motion.</p>
<p style="margin: 0 0 8px 0; color: #212529;"><strong>Essential vs. decorative motion:</strong></p>
<ul style="margin: 0 0 16px 0; padding-left: 24px; color: #212529;">
<li style="margin-bottom: 6px;"><strong>Essential:</strong> Progress indicators, state changes, drag-and-drop feedback</li>
<li><strong>Decorative:</strong> Hover effects, page transitions, background animations</li>
</ul>
<div style="display: flex; flex-wrap: wrap; gap: 16px; margin-bottom: 16px;">
<div style="flex: 1; min-width: 200px; background-color: #f8f9fa; border-radius: 6px; padding: 16px;">
